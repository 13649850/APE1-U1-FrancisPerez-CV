# Mi CV - Francis Pérez
En este proyecto se elaboró una página web que muestra mi CV, creado con HTML, CSS y JavaScript aplicando conceptos de diseño web, estructura semántica y manipulación del DOM con las siguientes funcionalidades:
- Secciones de perfil, formación, experiencia, proyectos, habilidades técnicas y blandas.
- Botones para mostrar y ocultar las habilidades, al igual que también para cambiar la pagina de color claro a oscuro y viceversa.
- Diseño adaptable a distintos dispositivos.
